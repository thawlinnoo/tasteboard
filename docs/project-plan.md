# TasteBoard Project Plan

## Project Idea

TasteBoard is a Flask web app where users can share restaurant posts, comment on restaurants, search by restaurant name or city, and allow admins to moderate posts/comments.

## MVP Features

- User register/login/logout
- First registered user becomes admin
- Users can create restaurant posts
- Users can view all restaurant posts
- Users can comment on posts
- Users can search by restaurant name or city
- Admin can delete any post
- Admin can delete any comment

## Out of Scope for MVP

- Real image upload
- Google Maps API
- Restaurant external API
- Payment/reservation system
- Real-time chat
